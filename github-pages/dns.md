# GitHub Pages (documentation)

The Nextra documentation site is published from the `docs` repository to GitHub Pages. This is separate from the live storefront, which runs on **AWS EC2** (`http://34.227.190.40`).

## Current URL

`https://gotokart.github.io/docs/`

## Optional custom domain

1. Purchase a domain from a registrar.
2. For GitHub Pages apex, add DNS **A** records to GitHub’s IPs:

   - `185.199.108.153`
   - `185.199.109.153`
   - `185.199.110.153`
   - `185.199.111.153`

3. In the **docs** repo: **Settings → Pages**, set the custom domain and enable HTTPS.
